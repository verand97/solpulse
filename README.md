# ⚡ SolPulse — Solana DEX Analyzer

Real-time Solana DEX analyzer with charts, whale tracking, and liquidity monitoring.

## Features

- **DEX Screener** — Real-time token pairs with price, volume, liquidity, and market cap
- **Portfolio Dashboard** — Track connected wallet assets and P&L performance
- **Whale Watcher** — Live alerts for large buy/sell transactions
- **Wallet & Swap** — Quick token swap interface with live price conversion
- **Search** — Instant token search across all pairs

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4
- Recharts
- Lucide Icons
- Framer Motion
